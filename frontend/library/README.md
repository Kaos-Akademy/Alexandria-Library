## Alexandria Library frontend

This folder contains the main Alexandria Library frontend, built with React, TypeScript, and Next.js.

### Development

- **Install dependencies**: run `npm install` from the repository root.
- **Start dev server**: run `npm run dev` from the repository root (this delegates to the app in this folder).

### Build & production

The app uses the standard Next.js build commands defined in `package.json`:

- `next build` for production builds
- `next start` to run the production server
